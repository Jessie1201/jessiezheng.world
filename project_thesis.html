M.Sc. Thesis

Interactive Visual Analytics for Agent-Based Simulation: Pedestrian Behavior at Signalized Pedestrian Crossing

This thesis is currently undergoing, which is the most exciting project that I ever did.


The objective application is a web-based data visualization tool that simulates the pedestrian behavior of running a red light. Concretely, building on top of PIM, the application models how individual interacts with each other & with the vehicle when the pedestrian flow rate changes. Deck.gl, D3, and C++ RVO2 simulation are heavily used.
